# Investor Due Diligence

## Accurate Claim

## Do Not Claim

- Production UPI integration.
- Live NPCI API access.
- Bank-grade security certification.
- Real customer data processing.
- Autonomous production AI decisioning.

- Full-stack React + Express implementation.
- Signed local demo-token RBAC workflow.
- CRUD APIs and frontend drill-down flows.
- Payment ecosystem simulator with five adapters.
- Webhook HMAC, duplicate, and out-of-order handling.
- Backend tests and local browser E2E smoke tests.
- Docker, CI workflow, deployment docs, SDLC docs, and model governance artifacts.
